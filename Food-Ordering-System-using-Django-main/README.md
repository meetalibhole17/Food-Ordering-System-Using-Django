# Food Ordering System using Django

This Django project implements a food ordering system where users can browse menu items, place orders, and manage their accounts.

## Features

- **User Authentication:** Registration, login, and logout functionalities for users.
- **Menu Management:** Admin can add, edit, and delete menu items.
- **Order Placement:** Users can add items to cart, place orders, and view order history.
- **Payment Integration:** Integration with payment gateways (if applicable).
- **User Profiles:** Users can update their profiles and view order history.
- **Admin Panel:** Secure admin interface to manage menu items, orders, and user accounts.

## Technologies Used

- Django
- HTML/CSS/JavaScript
- SQLite
